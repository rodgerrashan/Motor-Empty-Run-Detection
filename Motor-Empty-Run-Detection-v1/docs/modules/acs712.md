# Acs712Sensor module

Files:

- `include/sensors/Acs712Sensor.h`
- `src/Acs712Sensor.cpp`

## Purpose

- Read current magnitude using ACS712 analog output.
- Calibrate the **zero-current offset** at startup.

## How it works

- Samples ADC repeatedly.
- Converts to volts.
- Subtracts the offset.
- Computes RMS of the centered voltage, then divides by sensitivity (V/A).

## Configuration

- `ACS_SENS_V_PER_A` in `include/AppConfig.h`
- `ACS_VOUT_TO_ADC_SCALE` if you added a divider on ACS Vout.

## Calibration

- Make sure motor is OFF during the offset calibration period right after boot.

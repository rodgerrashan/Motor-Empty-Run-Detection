# Mpu6050Sensor module

Files:

- `include/sensors/Mpu6050Sensor.h`
- `src/Mpu6050Sensor.cpp`

## Purpose

- Initialize MPU6050 via I2C.
- Read accelerometer and internal temperature.
- Produce a simple demo-oriented `vibration_hz` estimate.

## Vibration estimate

`estimateVibrationHz()` is not a lab-grade frequency estimator.

It:

- Reads accelerometer magnitude over a short window
- Applies a simple high-pass filter
- Counts sign changes to estimate an approximate oscillation rate

This is sufficient for a UI demo where vibration increases when the motor runs.
